# পড়াশোনা সহায়ক - Frontend

HSC ও SSC শিক্ষার্থীদের জন্য AI-চালিত স্টাডি প্ল্যাটফর্মের ফ্রন্টএন্ড অ্যাপ্লিকেশন।

## প্রযুক্তি ব্যবহৃত হয়েছে
- Next.js (App Router) + TypeScript
- Tailwind CSS
- TanStack Query
- Framer Motion

## মূল ফিচারসমূহ
- JWT ও Google OAuth authentication (ডেমো লগইনসহ)
- নোট শেয়ারিং, ফিল্টারিং, সার্চ, রেটিং
- AI ফ্ল্যাশকার্ড/MCQ জেনারেটর (স্কোর ট্র্যাকিংসহ)
- স্ট্রিমিং AI টিউটর চ্যাট
- ব্যক্তিগত ড্যাশবোর্ড

## ইনস্টলেশন ও রান
```bash
npm install
npm run dev
```
অ্যাপ চালু হবে http://localhost:3000 এ।
