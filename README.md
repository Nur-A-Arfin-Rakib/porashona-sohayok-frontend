# 📚 পড়াশোনা সহায়ক — Frontend

HSC ও SSC শিক্ষার্থীদের জন্য AI-চালিত স্টাডি প্ল্যাটফর্মের ফ্রন্টএন্ড অ্যাপ্লিকেশন।

## 🛠️ টেক স্ট্যাক
- Next.js (App Router) + TypeScript
- Tailwind CSS
- TanStack Query
- Framer Motion

## ✨ মূল ফিচারসমূহ

### Authentication
- ইমেইল/পাসওয়ার্ড রেজিস্ট্রেশন ও লগইন
- Google OAuth Social Login
- ডেমো অ্যাকাউন্ট লগইন
- প্রোটেক্টেড রুট

### নোট শেয়ারিং সিস্টেম
- নোট আপলোড (ছবি ও ফাইল লিংকসহ)
- ক্লাস ও সাবজেক্ট ফিল্টার, সার্চ, সর্টিং
- স্টার রেটিং, রিভিউ ও আপভোট
- ব্যক্তিগত নোট ম্যানেজমেন্ট পেজ

### AI ফিচার
- AI ফ্ল্যাশকার্ড ও MCQ জেনারেটর (স্কোর ট্র্যাকিংসহ)
- স্ট্রিমিং AI টিউটর চ্যাট (ফলো-আপ সাজেশনসহ)

### UI/UX
- সম্পূর্ণ রেসপন্সিভ ডিজাইন
- স্মুথ পেজ ট্রানজিশন ও অ্যানিমেশন
- স্কেলিটন লোডার

## ⚙️ এনভায়রনমেন্ট ভ্যারিয়েবল

.env.local.example কপি করে .env.local বানাও এবং বসাও:

NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id

## 🚀 ইনস্টলেশন ও রান

npm install
npm run dev

অ্যাপ চালু হবে http://localhost:3000 এ।
